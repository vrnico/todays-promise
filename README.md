# Get Your Page Live · Follow Along

Work at your own pace. When your page is live, drop the link in chat.

---

## 1. Make a GitHub account

Go to **github.com** and sign up. It's free and takes about a minute.

Shout your username in chat when you're in.

---

## 2. Make your repository

1. Top-right **+** → **New repository**.
2. Repository name: name your repo what you want your project to be called (tentatively if you don't have an idea yet)
3. Set it to **Public**.
4. Check **Add a README file**.
5. Click **Create repository**.



---

## 3. Add your page

1. In your repo: **Add file** → **Create new file**.
2. Name the file **exactly** `index.html` (all lowercase).
3. Scroll down → **Commit changes** → commit to **main**.

That just saved a website to the internet.

---

## 4. Open your live URL

1. Open a new tab.
2. Go to **`yourname.github.io/your-project`**.
3. First load takes about a minute. If you see a **404**, don't panic. Wait and refresh.

When it loads, **drop your link in chat.**

---

## 5. Make it yours

Your page still says "Working title." Now make it your own **commit #1**. Both prompts are in chat:

- **Don't know your idea yet?** Run the **first prompt**. It asks you five questions, then gives you three ideas and the commit #1 for each.
- **Know your idea?** Run the **second prompt**. It writes your `index.html`. Paste it back into your repo (same as Step 3), commit, refresh.

You're swapping words and pasting code, not writing from scratch. Every time your page changes, re-post your link.

---

## Stuck?

- **404 that won't go away.** Give it a minute after committing. Check **Settings → Pages**, it should say "Your site is live."
- **Page won't publish.** Your repo name has a typo or a capital letter, or it's set to Private. Fix the name in **Settings**, and make sure it's **Public**.
- **Wrong file name.** It must be exactly `index.html`, not `Index.html` or `index.html.txt`. Rename it.
- **Page not updating.** GitHub takes about a minute after each commit. Hard-refresh the tab.
- **Still stuck?** Shout in chat. That's what this hour is for.

---

**The whole thing:** account → repo `you.github.io` → `index.html` (paste starter) → open the URL → make it yours.
